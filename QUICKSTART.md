# Soft Restaurant

Sistema completo de gestión para restaurantes con Flutter.

## Quick Start

```bash
# 1. Instalar dependencias
flutter pub get

# 2. Generar código
flutter pub run build_runner build --delete-conflicting-outputs

# 3. Configurar Firebase
flutterfire configure

# 4. Ejecutar
flutter run
```

Ver [README.md](README.md) completo para instrucciones detalladas.
